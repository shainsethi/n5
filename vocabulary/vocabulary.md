# Vocabulary (語彙)

Below are the Japanese vocabulary words for N5 level:

---

## Seion (清音) Practice

*(Words using only basic, unmodified kana)*

| Romaji | Hiragana | Meaning |
| :--- | :--- | :--- |
| kumo | くも | Cloud / Spider |
| sankaku | さんかく | Triangle |
| seikatsu | せいかつ | Life / Living / Livelihood |
| sensei | せんせい | Teacher |
| takenoko | たけのこ | Bamboo shoot |
| hotaru | ほたる | Firefly |
| taiikukan | たいいくかん | Gymnasium |
| saikoro | さいころ | Dice |
| omoshiroi | おもしろい | Interesting / Amusing |

---

## Sokuon (促音) Practice

*(Words containing a small tsu `っ` that doubles the following consonant)*

| Romaji | Kana | Meaning |
| :--- | :--- | :--- |
| torakku | トラック | Truck |
| roketto | ロケット | Rocket |
| koppu | コップ | Cup / Glass |
