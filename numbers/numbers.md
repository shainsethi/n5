# Numbers (数字 - Sūji)

Japanese numbers follow a very logical pattern, but there are important irregular readings and a different scaling system (Myriad) to master for JLPT N5.

## Basic Numbers (1-10)

| Number | Kanji | Hiragana | Romaji |
| :--- | :--- | :--- | :--- |
| **0** | 零 / 〇 | れい / ぜろ | rei / zero |
| **1** | 一 | いち | ichi |
| **2** | 二 | に | ni |
| **3** | 三 | さん | san |
| **4** | 四 | し / よん | shi / yon |
| **5** | 五 | ご | go |
| **6** | 六 | ろく | roku |
| **7** | 七 | しち / なな | shichi / nana |
| **8** | 八 | はち | hachi |
| **9** | 九 | きゅう / く | kyū / ku |
| **10** | 十 | じゅう | jū |

---

## Formation Rules (11-99)

Forming numbers between 11 and 99 is straightforward addition and multiplication:

- **11-19**: `10 + [digit]`
  - *Example*: 11 is **十一** (jū-ichi)
- **20, 30, ... 90**: `[digit] * 10`
  - *Example*: 20 is **二十** (ni-jū)
- **Combined**: `[digit] * 10 + [digit]`
  - *Example*: 25 is **二十五** (ni-jū-go)

## Large Numbers & Irregular Readings

When moving into hundreds and thousands, pay close attention to **sound changes** (Rendaku).

### Scaling Units
| Value | Kanji | Hiragana | Romaji |
| :--- | :--- | :--- | :--- |
| **100** | 百 | ひゃく | hyaku |
| **1,000** | 千 | せん | sen |
| **10,000** | 万 | まん | man |

### Common Irregulars
> [!IMPORTANT]
> These are the most common "gotchas" in JPLT N5 listening and reading.

| Number | Kanji | Reading | Note |
| :--- | :--- | :--- | :--- |
| 300 | 三百 | さん**びゃく** (san-byaku) | `h` → `b` |
| 600 | 六百 | **ろっぴゃく** (rop-pyaku) | Double consonant `p` |
| 800 | 八百 | **はっぴゃく** (hap-pyaku) | Double consonant `p` |
| 3,000 | 三千 | さん**ぜん** (san-zen) | `s` → `z` |
| 8,000 | 八千 | **はっせん** (has-sen) | Double consonant `s` |

## The Myriad System (4-Digit Units)

Unlike the Western system which groups by 3 digits (thousands), Japanese groups by **4 digits (万 - 10,000)**. 

### How to Convert
To read large Japanese numbers, mentally place a comma every **four** digits instead of three.

- **12,000** $\rightarrow$ `1,2000`
  - **一万二千** (ichi-man ni-sen)
- **100,000** $\rightarrow$ `10,0000`
  - **十万** (jū-man)
- **6,023,080** $\rightarrow$ `602,3080`
  - **六百二万三千八十** (rop-pyaku ni-man san-zen hachi-jū)

## Special Rules

### Silent Zero
> [!TIP]
> If `0` is in the middle of a number (like the hundreds or tens place), it is **not pronounced**.
> 
> *Example*: **409** is **四百九** (*yon-hyaku-kyū*). You do NOT say "zero" for the tens place.
