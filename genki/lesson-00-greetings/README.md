# Lesson 0 – Greetings (あいさつ)

---

## Everyday Greetings

| Kana | Romaji | Meaning |
| :--- | :--- | :--- |
| おはよう。 | Ohayoo. | Good morning. |
| おはよう ございます。 | Ohayoo gozaimasu. | Good morning. (polite) |
| こんにちは。* | Konnichiwa.* | Good afternoon. |
| こんばんは。* | Konbanwa.* | Good evening. |
| さようなら。 | Sayoonara. | Good-bye. |
| おやすみ（なさい）。 | Oyasumi (nasai). | Good night. |

> [!NOTE]
> *The last syllable of *konnichiwa* and *konbanwa* is spelled with **は** instead of **わ**.

## Gratitude & Apologies

| :--- | :--- | :--- |

## Coming & Going

| Kana | Romaji | Meaning |
| :--- | :--- | :--- |
| いってきます。 | Itte kimasu. | I'll go and come back. |
| いってらっしゃい。 | Itterasshai. | Please go and come back. |
| ただいま。 | Tadaima. | I'm home. |
| おかえり（なさい）。 | Okaeri (nasai). | Welcome home. |
